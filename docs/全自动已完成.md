## 已自动运行（每天凌晨 + 刚 push 后立即）

| 自动化 | 做什么 |
|--------|--------|
| **全站冒烟测试** | 56 站生产 URL 是否正常 |
| **SEO 扫描** | robots/sitemap/title/OG/JSON-LD → 写入 `analytics/rollup.json` |
| **IndexNow 提交** | 全站 URL 推送给 **Bing / Yandex**（加速收录，无需 Google 账号） |
| **GitHub 自动 commit** | 扫描结果写入仓库，看板可读 |
| **飞书通知** | 若已配 `FEISHU_WEBHOOK_URL`，每天发 SEO 均分摘要 |

工作流：`.github/workflows/seo-daily.yml`（也可在 GitHub Actions 里手动点 **Run workflow**）

## 关于 Google Search Console

Google **必须用你的 Google 账号登录**验证网站所有权，任何 Agent 都无法代替你登录 Google。

已通过 **IndexNow** 间接帮助 Bing 等搜索引擎收录；Google 通常会在数周内自行抓取 sitemap。

若以后你想加 Google：
- 只需在 [Search Console](https://search.google.com/search-console) 加 **1 个**你最关心的站
- 提交 `sitemap.xml` 即可  
**不是必须**，不影响站点正常运行。
