## 仅当看板一直显示「暂无数据」时（可选 1 步）

说明 Vercel 上的看板还没权限写 GitHub 数据文件。做一次即可：

1. 打开 https://github.com/settings/tokens → **Generate new token (classic)**
2. 勾选 **repo**，生成并复制 token
3. 打开 https://vercel.com → 项目 **factory-dashboard** → **Settings** → **Environment Variables**
4. 添加：
   - `GITHUB_TOKEN` = 刚复制的 token
   - `GITHUB_REPO` = `Waitind123/daily-sites-factory`
5. **Deployments** → 最新一条 → **Redeploy**

完成。之后数据会持久保存到仓库 `analytics/rollup.json`。
