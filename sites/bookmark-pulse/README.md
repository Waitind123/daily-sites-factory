# Feature Vote

Canny alternative for indie hackers — feature voting boards, public roadmap, $9.9/mo flat.

## Core features

- Create public feedback boards
- Users submit ideas and upvote (no login)
- Roadmap columns: Open → Planned → In Progress → Shipped
- 5 free boards, then $9.9/mo subscription
- `/join` pricing · `/api/checkout` · `/api/trial`

## Dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
